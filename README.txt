
## Installation

After enabling this this module as normal you need to do the following
on the configuration screen of the module:

1. Add an admin API Token from HipChat and the default room to receive
   messages at [your HipChat site]/account/api in order to enable user
   management features.
2. Add a room-specific API token from a custom integration to enable
   sending messages. You can create a custom integration at 
   [your HipChat site]/admin/byo
